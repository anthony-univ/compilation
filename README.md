# M1 ISL Project Compilation
This project is developed as part of the first semester of MASTER 1 ISL in Besancon.
To develop this software, we were able to use the IntelliJ IDE. We were able to learn about agile practices using for example Trello, planning poker, burdownChart,...

## Functionality
- Edit Minijaja files with an IDE 
- Interpret MiniJaja files
- Compile Minijaja files into JajaCode

## Languages used 
- JAVA (JAVAFX)
- JAVACC
- CSS
